# AlphaPulse AI - Landing Page (Standalone)

صفحة هبوط منفصلة تماماً يمكن تشغيلها بشكل مستقل على أي جهاز أو استضافة

## مميزات الصفحة:

✅ **مستقلة تماماً** - بدون dependencies محلية
✅ **سهل التشغيل** - افتح `index.html` في أي متصفح
✅ **سريعة الرفع** - يمكن نشرها على أي استضافة في ثواني
✅ **متجاوبة** - تعمل على جميع الأجهزة (جوال، تابلت، ديسكتوب)
✅ **محتوى كامل** - كل CSS و JavaScript مدمج في ملف واحد

## طريقة التشغيل:

### على جهازك المحلي:
```bash
# 1. انسخ مجلد landing-page كاملاً
# 2. افتح الملف:
open landing-page/index.html    # Mac
start landing-page/index.html   # Windows
# أو اضغط مزدوج على index.html
```

### على استضافة ويب:
```bash
# 1. رفع مجلد landing-page كاملاً
# 2. الصفحة ستعمل فوراً - لا تحتاج أي setup
```

## المميزات الإضافية:

- **عداد زوار حي** - يزيد كل 5 ثواني
- **إشعارات مباشرة** - تظهر عشوائياً من مدن مختلفة
- **أزرار فعالة** - جميع الروابط تعمل بسلاسة
- **نسخ العنوان الذاتي** - انسخ عنوان المحفظة بكلك واحد
- **متوافق مع جميع الدول** - بدون مشاكل geo-blocking

## الملاحظات المهمة:

⚠️ **الصور الاختيارية**:
- إذا لم تجد الصور في مجلد `images`، الصفحة ستعرض placeholders
- يمكنك إضافة الصور لاحقاً في نفس المجلد

✅ **CDNs الخارجية** (وهي آمنة):
- Tailwind CSS (من CDN رسمي)
- Font Awesome Icons (من CDN رسمي)
- Google Fonts (Inter font)

## الدعم:
