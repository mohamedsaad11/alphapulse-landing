# 🎥 دليل إضافة الفيديو

## كيفية إضافة الفيديو إلى الصفحة

### 📍 المسار الصحيح للفيديو:

```
landing-page/
├── index.html
├── images/
│   ├── ... (جميع الصور)
│   └── ⭐ promo.mp4         👈 ضع الفيديو هنا!
└── VIDEO_INSTRUCTIONS.md
```

### ✅ الخطوات البسيطة:

#### **الخطوة 1: تجهيز الفيديو**
- استخرج الفيديو من أي مصدر (YouTube, Screencast, إلخ)
- تأكد من أن الصيغة: **MP4** (الأفضل)
- يمكن استخدام صيغ أخرى: WebM, MKV, AVI

#### **الخطوة 2: حفظ الفيديو**
```
احفظ باسم: promo.mp4
في المجلد: landing-page/images/
```

#### **الخطوة 3: تحديث الصفحة**
- فقط افتح index.html في المتصفح
- الفيديو سيظهر تلقائياً! ✨

### 🔧 معلومات تقنية:

| الخاصية | القيمة |
|--------|--------|
| **الصيغة المفضلة** | MP4 (.mp4) |
| **الصيغ المدعومة** | MP4, WebM, OGG, MOV |
| **الحجم المقترح** | أقل من 50MB |
| **الدقة** | 1920x1080 أو أقل |
| **معدل الإطارات** | 30fps أو 60fps |

### 🎬 مثال على الملف:

```
landing-page/images/promo.mp4        ✅ يعمل
landing-page/images/PROMO.MP4        ✅ يعمل
landing-page/images/video.mp4        ❌ لن يعمل (اسم خاطئ)
landing-page/video.mp4               ❌ لن يعمل (مسار خاطئ)
```

### ❓ ماذا يحدث عند الضغط على الفيديو؟

1. ✅ **الفيديو موجود** → يشغل الفيديو مباشرة
2. ❌ **الفيديو غير موجود** → ظهور رسالة توضيحية

### 🎯 هل تريد فيديو مختلف؟

ببساطة:
1. احذف `promo.mp4` القديم
2. ضع `promo.mp4` جديد
3. احدّث المتصفح (Refresh)

### 📊 حجم الفيديو المقترح:

| الطول | الحجم | الجودة |
|-------|-------|--------|
| 30 ثانية | 5-10 MB | عالية |
| 1 دقيقة | 10-20 MB | عالية |
| 2 دقيقة | 20-40 MB | عالية |

### 🛠️ أدوات مقترحة لضغط الفيديو:

**على الويب (مجاني):**
- HandBrake (https://handbrake.fr/)
- CloudConvert (https://cloudconvert.com/)
- TinyWow (https://tinywow.com/)

**على سطر الأوامر (لينكس/Mac):**
```bash
# ضغط بدون فقدان جودة كثيرة:
ffmpeg -i input.mp4 -vcodec h264 -crf 23 promo.mp4

# ضغط قوي:
ffmpeg -i input.mp4 -vcodec h264 -crf 28 promo.mp4
```

### 🚀 بعد إضافة الفيديو:

✅ الصفحة ستعمل بدون مشاكل
✅ الفيديو سيظهر فوراً عند فتح الصفحة
✅ يمكن تشغيل الفيديو بالضغط على الزر الأزرق
✅ يمكن التحكم في مستوى الصوت والإيقاف المؤقت

### ⚠️ المشاكل الشائعة وحلولها:

#### ❌ المشكلة: الفيديو لا يظهر
**الحل:**
- تأكد من اسم الملف: `promo.mp4` (بالضبط)
- تأكد من موقع الملف: `landing-page/images/`
- تأكد من صيغة الملف: MP4

#### ❌ المشكلة: الفيديو لا يشتغل
**الحل:**
- جرب صيغة مختلفة (WebM أو OGG)
- تأكد من أن الفيديو يعمل بشكل منفصل
- حاول ضغط الفيديو

#### ❌ المشكلة: الصفحة بطيئة
**الحل:**
- اضغط الفيديو أكثر
- استخدم ffmpeg مع CRF أعلى
- قلل دقة الفيديو

### 💡 نصائح احترافية:

1. **إضافة صورة مصغرة (Thumbnail):**
```html
<!-- في الصفحة، غيّر هذا السطر: -->
<video id="promo-video" ... poster="images/thumbnail.png">
```

2. **تعطيل التحميل المسبق:**
```html
<video id="promo-video" preload="none" ...>
```

3. **إضافة نسخة بديلة:**
```html
<video id="promo-video" ...>
    <source src="images/promo.mp4" type="video/mp4">
    <source src="images/promo.webm" type="video/webm">
</video>
```
